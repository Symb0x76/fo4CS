This folder is part of the packaged Fallout 4 Data layout.

amd_fidelityfx_dx12.dll is required for AMD FSR upscaling and AMD FSR frame generation.
The plugins load it from:

Data\F4SE\Plugins\FidelityFX\amd_fidelityfx_dx12.dll

Keep this DLL with the package and dist output when shipping FSR support.
